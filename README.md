## 📌 Featured Projects

### [Births, Beats & Basketball — Data Visualization Mini-Project 2](https://github.com/[your-username]/Will-Bowden_dataviz-mini-project-2)
An R Markdown report exploring three unrelated datasets — U.S. birth records, Billboard summer hits, and NBA Finals box scores — through five distinct visualizations.

- 📅 **U.S. Births (2000–2014):** Heatmap and bar chart revealing seasonal rhythms and a stark weekday/weekend gap (~32%) driven by hospital scheduling
- 🎵 **Billboard Summer Hits (1958–2017):** Interactive Plotly scatter plot showing how the emotional range of summer hits has widened dramatically since the 1990s
- 🏀 **NBA Champions (1980–2018):** Spatial map of championship geography and an OLS coefficient plot predicting per-game scoring (R² ≈ 0.68)
