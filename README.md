## Projects 📌

### [Births, Beats & Basketball](https://github.com/WillBowden8/Will-Bowden_dataviz-mini-project-2)

An R Markdown report exploring three unrelated datasets through five distinct visualizations:

- **U.S. Births (2000–2014)** — seasonal heatmap and day-of-week bar chart revealing how hospital scheduling shapes birth patterns
- **Billboard Summer Hits (1958–2017)** — interactive scatter plot tracing how the emotional tone of summer music has shifted over 60 years
- **NBA Champions (1980–2018)** — a geographic map of championship dominance and a regression model predicting per-game scoring
